# 基本形
```
docker run ()
```

# できたもの
- python
- php
- perl
- java
- nodejs
- golang
- haskell
- ruct

# できなかったもの
https://hub.docker.com/ で公開されているものを探せばできます。
一例ですが、

- scala
- kotlin
- rlang

などは比較的簡単に見つかります。

# 言語やスクリプト以外
- mysql
- postgres
- mongo
- mariadb

など、他多数。
wordpressなんかもofficialがありました。
