# partsについて
<a href="https://docs.python.jp/3/library/pathlib.html#accessing-individual-parts">公式ドキュメント：parts</a>に書いていますが、ちゃんと調査をしてみます。

.parts[-1:]
　　.name(basename)を取得したい
.parts[[:-1]
　　parent,parents[]を取得したい
.parts[[1:-1]
　　上から指定のパスを消し、下から指定のパスを消したい
.parts[[:1]
　　上から順番にパスを取得したい
.parts[[1:]
　　上から順番にパスを消したい

というようになるみたいです。
ややこしくて理解するのが大変でした。
